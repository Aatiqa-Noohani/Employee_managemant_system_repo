<h2>📌 Project Overview</h2>

<p>
The Employee Management System is a desktop application developed using Python Tkinter.
It allows users to manage employee records through a user-friendly graphical interface.
Employee data is stored permanently in a CSV file.
</p>

<h2>🚀 Features</h2>

<ul>
    <li>➕ Add Employee</li>
    <li>✏️ Update Employee Information</li>
    <li>🗑 Delete Employee Records</li>
    <li>🔍 Search Employee by ID</li>
    <li>📋 View Complete Employee Details</li>
    <li>💾 Save Data in CSV File</li>
    <li>🔄 Auto Load Existing Records</li>
    <li>🖥 Interactive GUI Interface</li>
</ul>
